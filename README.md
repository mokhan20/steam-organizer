# Vault - Steam Library Organizer

A desktop application built with Electron for organizing your Steam game library.

## Features
- Browse your full Steam library with cover art
- Filter games by hours played, last played date, and review score
- Create custom categories and drag games into them
- Click any game to view details and description

## Setup
1. Clone the repo
2. Copy `config.example.js` to `config.js` and fill in your Steam API key and Steam ID
3. Run `npm install`
4. Run `npm start`

## Tech
- Electron
- Steam Web API
- Vanilla JavaScript, HTML, CSS
